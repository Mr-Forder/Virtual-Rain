# Virtual Rain
Virtual Rain is a Gaming Magazine style news and review 
aggregator,  utilizing Rawg.io's API with state management via Redux.
Animated assets created via Lottie's Bodymovin AE plugin.




## To get Started

```
npm install

npm start
```
